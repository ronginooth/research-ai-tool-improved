# 月読み（Tsukuyomi）アイコンデザイン案

静けさをイメージした色合いのアイコンデザイン案です。

## デザインコンセプト

「月読み（Tsukuyomi）」は日本の月の神様に由来し、静けさ、知性、知識を象徴します。
- **静けさ**: 深い色合い、落ち着いた印象
- **月の光**: 柔らかな白いストローク
- **知識**: 本のアイコンで表現

## デザインの特徴

すべてのバリエーションで共通：
- **静けさを表現**: 深い色合いと柔らかな透明度
- **月の光**: 白いストロークに透明度を設定し、柔らかな印象
- **均等な間隔**: 横棒の間隔を均等に配置
- **円形フレーム**: 統一感のあるデザイン

## 使用方法

お気に入りのカラーを選択し、`src/app/icon.svg`にコピーしてください。

```bash
# 例: ミッドナイトブルーを採用する場合
cp src/app/icon-options/icon-tsukuyomi-1-midnight-blue.svg src/app/icon.svg
```
