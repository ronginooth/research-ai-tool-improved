# アイコンデザイン案

Research AI Tool 用の favicon アイコンのデザイン案です。

## デザイン案一覧

### 1. icon-1-simple-book.svg

- **スタイル**: シンプルな本のアイコン
- **色**: 青 (#3B82F6)
- **特徴**: クリーンで分かりやすいデザイン

### 2. icon-2-book-search.svg

- **スタイル**: 本 + 検索アイコン
- **色**: 紫 (#8B5CF6)
- **特徴**: 検索機能を強調

### 3. icon-3-book-ai.svg

- **スタイル**: 本 + AI（脳）アイコン
- **色**: 緑 (#10B981)
- **特徴**: AI 機能を強調

### 4. icon-4-book-pen.svg

- **スタイル**: 本 + ペンアイコン
- **色**: オレンジ (#F59E0B)
- **特徴**: ライティング機能を強調

### 5. icon-5-book-network.svg

- **スタイル**: 本 + ネットワークアイコン
- **色**: ピンク (#EC4899)
- **特徴**: 接続・ネットワーク機能を強調

### 6. icon-6-gradient.svg

- **スタイル**: グラデーション背景の本
- **色**: 青 → 紫のグラデーション
- **特徴**: モダンで視覚的に魅力的

### 7. icon-7-minimal.svg

- **スタイル**: ミニマルデザイン
- **色**: 濃い青 (#1E40AF)
- **特徴**: シンプルで洗練されたデザイン

### 8. icon-8-3d.svg

- **スタイル**: 3D 風デザイン
- **色**: インディゴ (#6366F1)
- **特徴**: 立体感のあるデザイン

### 9. icon-9-modern-flat.svg

- **スタイル**: モダンなフラットデザイン
- **色**: シアン (#06B6D4)
- **特徴**: 角丸とフラットなデザイン

## 使用方法

お気に入りのデザインを選択し、`src/app/icon.svg`にコピーしてください。

```bash
# 例: icon-1を採用する場合
cp src/app/icon-options/icon-1-simple-book.svg src/app/icon.svg
```
