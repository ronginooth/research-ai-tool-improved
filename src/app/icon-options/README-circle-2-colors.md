## 使用方法

お気に入りのカラーを選択し、`src/app/icon.svg`にコピーしてください。

```bash
# 例: 青を採用する場合
cp src/app/icon-options/icon-circle-2-blue.svg src/app/icon.svg
```

## デザインの特徴

すべてのバリエーションで共通：
- 円形フレーム（角丸の正方形）
- 曲線で表現した本のアイコン
- 均等な間隔の横棒（ページの曲線）
- 白いストロークで統一されたデザイン
