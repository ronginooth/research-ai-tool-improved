# 円と曲線を使ったアイコンデザイン案

Research AI Tool 用の favicon アイコン - 円と曲線を多用したデザイン案です。

## デザイン案一覧

### 1. icon-circle-1.svg

- **スタイル**: 円形の本 + 曲線のページ
- **色**: 青 (#3B82F6)
- **特徴**: シンプルな円形フレームに曲線で表現した本のページ

### 2. icon-circle-2.svg

- **スタイル**: 円形フレーム + 曲線の本
- **色**: 紫 (#8B5CF6)
- **特徴**: 円形の枠内に曲線で描かれた本

### 3. icon-circle-3.svg

- **スタイル**: 同心円 + 波線のページ
- **色**: 緑 (#10B981)
- **特徴**: 複数の円と波線で表現した本

### 4. icon-circle-4.svg

- **スタイル**: 流れるような曲線の本
- **色**: オレンジ (#F59E0B)
- **特徴**: 滑らかな曲線と装飾的な円の組み合わせ

### 5. icon-circle-5.svg

- **スタイル**: 波線のページ + 円形アクセント
- **色**: ピンク (#EC4899)
- **特徴**: 波打つような曲線と小さな円の装飾

### 6. icon-circle-6.svg

- **スタイル**: グラデーション円 + 滑らかな曲線
- **色**: 青 → 紫のグラデーション
- **特徴**: グラデーション背景に滑らかな曲線の本

### 7. icon-circle-7.svg

- **スタイル**: 多層円 + 流れる曲線
- **色**: インディゴ (#6366F1)
- **特徴**: 複数の円の層と流れるような曲線

### 8. icon-circle-8.svg

- **スタイル**: モダンな曲線デザイン
- **色**: シアン (#06B6D4)
- **特徴**: 洗練された曲線と円形アクセント

### 9. icon-circle-9.svg

- **スタイル**: 有機的な曲線 + 同心円
- **色**: ティール (#14B8A6)
- **特徴**: 有機的な流れを持つ曲線と複数の円

## 使用方法

お気に入りのデザインを選択し、`src/app/icon.svg`にコピーしてください。

```bash
# 例: icon-circle-1を採用する場合
cp src/app/icon-options/icon-circle-1.svg src/app/icon.svg
```
